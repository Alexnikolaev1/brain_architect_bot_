# 🧠 Архитектор Мозга — Telegram Bot

> *Нейрофитнес нового поколения. 5-7 минут в день — ощутимый результат через 21 день.*

---

## Быстрый локальный запуск (только с токеном)

1. Создай `.env` и добавь:
   - `TELEGRAM_BOT_TOKEN=...`
2. Установи зависимости:
   - `python -m pip install -r requirements.txt`
3. Запусти бота:
   - `python run_local.py`

### Smoke-проверка перед запуском

- Базовая: `python scripts/smoke_local.py`
- С проверкой токена через Telegram API: `python scripts/smoke_local.py --network`
- Принудительно с записью в внешний DB/Redis: `python scripts/smoke_local.py --write-checks`

### Важно

- Для локальной разработки `DATABASE_URL` (или `POSTGRES_URL`) и `UPSTASH_REDIS_URL` не обязательны.
- Если они не заданы, бот автоматически работает в `in-memory` режиме:
  - пользователи/статистика/сессии/лидерборд хранятся в памяти процесса;
  - после перезапуска эти данные очищаются.
- Для production используй Postgres + Redis.

## 5 Игр — Механики и Научное Обоснование

### ⚡ Игра 1: Двойной удар (`double_strike`)

**Научная база:** Нейробика Лоуренса Каца — одновременная активация моторной коры
и математических центров создаёт новые нейронные связи.

**Механика:**
```
Бот:  "2 + 3 = ?"
      🔴 5   🟢 7
      🔵 4   🟡 6

      🎯 Правило раунда:
      👏 Сначала хлопни в ладоши — потом жми!

Юзер: [нажимает 🔴 5]
Бот:  ✅ Верно! +200 XP
      ⏱ Время: 3.2с
      ⚔️ Прокачан архетип: Knight
```

**Сложность 1-3:** a+b или a-b, числа до 9
**Сложность 4-6:** до 50, включая умножение
**Сложность 7-10:** трёхзначные операции

### 🖐️ Игра 2: Лабиринт ощущений (`sensation_maze`)

**Научная база:** Нейропластичность Дойджа — мультисенсорная стимуляция
укрепляет ассоциативные сети между разными зонами коры.

**Механика:**
```
Бот:  "Представь, что ты держишь в руках:
       Мокрый 💧, холодный 🥶, гладкий,
       слегка скользкий предмет с мыльным запахом"

      ❓ Что это за предмет?

      🧼 Кусок мыла
      🧊 Кубик льда
      🪨 Морской камень
      🍦 Мороженое

Юзер: [выбирает 🧼 Кусок мыла]
Бот:  ✅ Верно!
      🧪 Мыло холодит (растворяясь, поглощает тепло),
         мокрое и гладкое из-за щелочной поверхности.
      🏹 Прокачан архетип: Scout
```

### 🏠 Игра 3: Анти-риэлтор (`anti_realtor`)

**Научная база:** Пространственное мышление (Shepard & Metzler, 1971) —
мысленное вращение и перемещение объектов активирует теменные и
лобные доли.

**Механика (2 фазы):**
```
ФАЗА 1 — Запоминание:
Бот:  "Запомни расстановку мебели:
       • Диван — у левой стены
       • Картина — над диваном
       • Торшер — справа от дивана
       • Стол — в центре комнаты
       • Полка — у правой стены

       ⏱ У тебя 20 секунд."

      [✅ Готов! Показывай изменение]

ФАЗА 2 — Найди изменение:
Бот:  "Новая расстановка:
       • Диван — у левой стены
       • Картина — над столом    ← изменение здесь!
       • Торшер — справа от дивана
       • Стол — в центре комнаты
       • Полка — у правой стены

       ❓ Какой предмет переставили?"

      Диван | Картина | Торшер | Стол

Юзер: [Картина]
Бот:  ✅ Верно! Картина переехала с дивана на стол.
```

### 🛑 Игра 4: Стоп-кран интуиции (`stop_signal`)

**Научная база:** Даниэль Канеман «Думай медленно, решай быстро» —
принудительное торможение Системы 1 и включение Системы 2.

**Механика:**
```
Бот:  "🏏 Мяч и бита стоят вместе 1 доллар 10 центов.
       Бита дороже мяча ровно на 1 доллар.

       Интуиция кричит: мяч стоит 10 центов.
       Почему это невозможно?

       Выбери ВЕРНОЕ объяснение:"

      A) Если мяч 10ц, бита = 1$10ц, итого 1$20ц ≠ 1$10ц
      B) Потому что 10 > 5 и это слишком много
      C) Нельзя иметь дробные центы
      D) Задача сформулирована некорректно

Юзер: [A]
Бот:  ✅ Верно!
      🧪 Пусть мяч = X. Бита = X + 1$.
         X + (X + 1$) = 1$10ц → 2X = 10ц → X = 5ц.
         Интуиция не сделала алгебраический шаг.
      🔮 Прокачан архетип: Oracle
```

ФАЗА 2 — Ввод (слова скрыты):
Бот:  "⌨️ Слова скрыты!
       Напечатай через пробел в обратном порядке."

Юзер: архитектура воображение концентрация алгоритм нейропластичность

Бот:  💯 Идеально!
      Правильно: архитектура воображение концентрация алгоритм нейропластичность

## Пошаговый деплой на Vercel

### 1. Подготовка сервисов

**Postgres (Vercel Postgres / Neon):**
```bash
# 1. В Vercel: Storage → Create Database → Postgres
# 2. Либо в Neon: создайте проект и базу
# 3. Скопируй connection string (postgresql://...sslmode=require)
```

**Upstash Redis (бесплатный план):**
```bash
# 1. Зарегистрируйся на upstash.com
# 2. Create Database → Redis → Free
# 3. Скопируй Endpoint и Password
```

**Telegram Bot:**
```bash
# 1. Открой @BotFather
# 2. /newbot → задай имя и username
# 3. Скопируй токен
# 4. /mybots → выбери бота → Payments → Telegram Stars (включить)
```

### 2. Деплой на Vercel

```bash
# Установи Vercel CLI
npm i -g vercel

# Клонируй и войди в директорию
cd brain_architect

# Первый деплой
vercel

# Добавь секреты
vercel env add TELEGRAM_BOT_TOKEN
vercel env add DATABASE_URL
vercel env add UPSTASH_REDIS_URL
vercel env add UPSTASH_REDIS_PASSWORD

# Задеплой с переменными
vercel --prod
```

### 3. Регистрация Webhook

```bash
# Получи URL деплоя (вида your-project.vercel.app)
python scripts/set_webhook.py https://your-project.vercel.app/api/webhook

# Проверь
python scripts/set_webhook.py
```

### 4. Создание индексов Postgres

```sql
-- Обычно создаются автоматически при старте приложения.
-- При ручной инициализации можно выполнить:
CREATE INDEX IF NOT EXISTS idx_game_stats_user_played
ON game_stats(user_id, played_at DESC);

CREATE INDEX IF NOT EXISTS idx_game_stats_game_user
ON game_stats(game_id, user_id);
```

### 5. Ограничения Vercel и их решения

| Проблема | Решение |
|---|---|
| Таймаут 25с (Pro) / 10с (Hobby) | Все операции < 5с: Redis + Postgres async |
| Stateless функции | Состояние в Redis (session) и Postgres (users/game_stats) |
| Cold start ~500ms | Переиспользуем `_application` singleton |
| Нет cron на Hobby | Используй Vercel Cron (Pro) или GitHub Actions |
